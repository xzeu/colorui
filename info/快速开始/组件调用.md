# 组件调用

## ColorUI组件位置

```
|_ colorui
    |_components
        |_组件列表
```

## 1、调用说明
1.1 `js` 中引用组件以`banner`为例

```js
import banner from '../../colorui/components/banner.vue';
export default {
    components : {
        banner
    }
}

```

1.2 在模板中使用组件

```html
<banner></banner>
```
## 2、全局注册某个组件

2.1 在`main.js` 中注册的组件可以在全局使用:

```js
import banner from "./hdpUI/components/banner.vue";
Vue.component('banner', banner);
```
2.2 在模板中使用组件

```html
<banner></banner>
```
